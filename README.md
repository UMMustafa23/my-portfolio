# ULVIE.EXE — Portfolio

> ACME DETECTIVE AGENCY · OPERATIVE DOSSIER · REF# UM-2026

Personal portfolio of **Ulvie Mustafa** — a student developer from Burgas, Bulgaria, studying IT & Software Development at VSCPI. Styled as a black-and-white retro detective agency terminal, inspired by *Where in the World Is Carmen Sandiego?*

## 🕵️ Features

- **Boot sequence** — CRT-style loading screen with typewriter terminal output
- **Operative dossier** — hero section with a classified personnel file and glitch effects
- **Case files** — interactive project browser with screenshots, evidence lightbox, and repo links
- **Scroll-spy navigation** — the navbar follows the section you're reading
- **Retro sound design** — mechanical clicks, hover blips, and boot chimes (mutable)
- **RPG stat bars** — skills and language proficiency rendered as game stats

## 🚀 Running Locally

```bash

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and press any key when the terminal finishes booting.
